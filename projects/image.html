Wildlife Image Classification

This project demonstrates the application of multiple pre-trained models—MobileNetV2, ResNet50, InceptionV3, and EfficientNetB0—for classifying images of various wildlife species. Leveraging the power of transfer learning, it efficiently categorizes images using TensorFlow and Keras, with model performance evaluated and compared to identify the most effective architecture.

Tech Stack

  • Python, Jupyter Notebook
  • TensorFlow, Keras
  • Pre-trained Models: MobileNetV2, ResNet50, InceptionV3, EfficientNetB0
  • Libraries: NumPy, Matplotlib

Dataset

The dataset contains images of various wildlife species divided into training, validation, and test sets. Each model was fine-tuned on this dataset to enhance performance and reduce training time.

Training & Evaluation

Each model was trained and evaluated using standard metrics such as accuracy and loss. The results were analyzed and visualized to determine the best-performing model for this task.

Customization Options

  • Swap out base model (e.g., use DenseNet or VGG instead)
  • Modify or expand classifier head architecture
  • Tune hyperparameters: learning rate, batch size, epochs
  • Apply custom preprocessing and data augmentation
