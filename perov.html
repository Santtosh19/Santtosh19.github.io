Perovskite Material Insights Platform

Live Dashboard ↗

This platform explores the properties of Cs₂NaTlBr₆₋ₓClₓ mixed halide double perovskites using DFT simulation data. Users can visualize how electronic, optical, and mechanical properties evolve with varying anion compositions (x = 0 to 6).

  • Interactive DFT Viewer: Band structures, DOS, optical spectra per composition.
  • Surrogate Modeling: Predict intermediate properties using Polynomial & GPR models.
  • Trade-Off Explorer: Interactive 3D tool to balance competing material properties.

Data Engineering

Created a pipeline to parse raw DFT outputs into a clean, queryable SQLite DB. Used custom parsers to automate validation and structure.

Machine Learning

Trained surrogate models (Polynomial Regression & Gaussian Process) to estimate Direct Bandgap, Bulk Modulus, and Debye Temperature for untested x compositions.

Tech Stack

  • Python 3.9+, Pandas, NumPy, SQLite
  • Scikit-learn, Joblib, Plotly, Streamlit
  • Custom parsing using os, re, pathlib

How to Run

  1. Clone the repo: git clone https://github.com/Santtosh19/perovskite-data-platform
  2. Run data pipeline: Python scripts in ingestion_scripts/
  3. Train models: Run notebooks/02_Surrogate_Modeling.ipynb
  4. Launch app: streamlit run app/main_dashboard.py
