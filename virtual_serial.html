High-Fidelity Reliability Monitoring Pipeline

This project simulates a complete Site Reliability Engineering (SRE) system by emulating a hardware device that streams temperature, voltage, and status data through a virtual serial port. The system performs real-time ingestion, structured logging, anomaly detection, and structured alert generation in JSON format.

  • Device Emulator: Simulates operational states (NORMAL, WARNING, CRITICAL, OFFLINE) via serial stream.
  • Data Logger: Parses, validates, and stores both raw and clean data logs (CSV + JSON).
  • Anomaly Detector: Identifies threshold breaches, rapid rate changes, and heartbeat losses.
  • Alerts: Generates machine-readable alerts in anomaly_report.json.

üíª Local Setup

  1. Create virtual ports COM5 <-> COM6 using Eltima or com0com.
  2. Run python emulator.py (simulated device output).
  3. Run python logger_service.py (data ingestion).
  4. Run python detector.py (analysis and alerts).
