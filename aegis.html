Aegis: A Resilient, Autonomous DataOps Framework

Aegis is a professional-grade, end-to-end data engineering project that demonstrates advanced skills in ETL, automation, testing, and observability — all integrated into a resilient pipeline.

Core Problem Solved

It tackles the real-world challenge of turning messy, unreliable telemetry data into clean, trusted analytics-ready output through a fully tested and automated pipeline.

Key Features & Skills Demonstrated

  • Data Engineering: High-throughput PySpark ETL, 1M+ records
  • Data Quality Gateway: PySpark-native validation engine to enforce data contracts
  • CI/CD Automation: GitHub Actions for linting, scanning, and testing
  • Testing & QA: Unit + property-based testing with Codecov integration
  • Observability: Structured logs, performance summaries, test matrices
  • Deployment: Auto-publish to TestPyPI via CD on version tag
  • Professional Tooling: Ruff, Black, Hydra, Trufflehog, Pre-commit

How to Run

To run locally, set up with Python 3.9, Java, Hadoop winutils, then:

# Clone repo
$ git clone https://github.com/Santtosh19/aegis_dataops_pipeline.git
$ cd aegis_dataops_pipeline

# Set Python version and create virtual env
$ pyenv local 3.9.13
$ python -m venv venv
$ source venv/bin/activate  # Windows: .\venv\Scripts\activate

# Install dependencies
$ pip install ".[dev]"

# Generate raw data
$ python scripts/generate_data.py

# Run the pipeline
$ python aegis/pipeline.py

Result: Clean data lands in data/processed/, and invalid records are quarantined in data/quarantined/.

Using Aegis as a Package

Install from TestPyPI to use core validation logic in your own project:

pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple aegis-framework

Example:

from pyspark.sql import SparkSession
from pyspark.sql.functions import col
from aegis.pipeline import validate_and_separate_data

spark = SparkSession.builder.getOrCreate()
sensor_df = spark.read.json("path/to/data.json")

rules = {
  "missing_id": sensor_df.filter(col("id").isNull()),
  "invalid_pressure": sensor_df.filter(col("pressure_psi") < 0)
}

clean_df, quarantined_df = validate_and_separate_data(
    df=sensor_df,
    rules=rules
)
