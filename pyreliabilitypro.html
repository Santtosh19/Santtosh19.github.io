PyReliabilityPro

A Python Toolkit for Modern Reliability Analysis

A lightweight open-source toolkit designed for engineers and analysts to conduct statistical reliability calculations, built with test-driven development and CI/CD automation.

View on GitHub

Key Features

  • Weibull Analysis: Estimate parameters, compute CDF, PDF, SF, and HF with real failure data.
  • MTTF Calculations: Both theoretical and sample-based for exponential & Weibull distributions.
  • CI/CD: Fully automated pipeline using GitHub Actions (lint, format, test, codecov upload).
  • 95%+ Test Coverage: Pytest-powered, tested against multiple Python versions.
  • Quality Assurance: Integrated flake8, black, and mypy checks on every commit.

Example Usage

import pyreliabilitypro as rel

failure_times = [105, 120, 135, 160, 175, 190, 210, 230, 255, 280]
beta, eta, _ = rel.weibull_fit(failure_times)

print(f"MTTF: {rel.weibull_mttf(beta, eta):.2f} hours")
print(f"Failure Probability @150h: {rel.weibull_cdf(150, beta, eta):.2%}")
print(f"Hazard Rate @150h: {rel.weibull_hf(150, beta, eta):.4f}")

Note

This project is designed for terminal or programmatic usage and does not include a graphical UI. Screenshots are not applicable for this type of engineering package.
